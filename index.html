📋 Pasos para conectar

  1. Abre tu Google Sheet
  2. Ve a Extensiones → Apps Script
  3. Borra el código existente y pega el nuevo
  4. Clic en Guardar (ícono disco)
  5. Clic en Implementar → Nueva implementación
  6. Tipo: Aplicación web
  7. Ejecutar como: Yo
  8. Acceso: Cualquiera
  9. Clic en Implementar → Autorizar → Avanzado → Continuar
  10. Copia la URL y pégala arriba ↑
